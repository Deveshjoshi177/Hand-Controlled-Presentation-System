# Hand Controlled Presentation System

## Description
This project allows users to control presentation slides using hand gestures.
It uses computer vision and hand tracking to detect gestures and perform slide navigation.

## Technologies Used
- Python
- OpenCV
- MediaPipe
- PyAutoGUI

## Features
- Next slide using two fingers
- Previous slide using one finger
- Real-time hand tracking
- No physical input devices required

## How to Run
```bash
pip install -r requirements.txt
python hand_controlled_presentation.py
